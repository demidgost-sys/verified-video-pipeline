# Asset provenance

No real footage or production asset is included in this repository.

The acceptance test creates a temporary audiovisual fixture at runtime with
FFmpeg's `testsrc2` and `sine` generators. The fixture is intentionally ignored
by Git and deleted with its temporary test directory. Its purpose is technical
verification only; it depicts no person, brand, course, channel, or customer.

Future assets are accepted only when this file records their creator, source,
license, allowed use, and SHA-256 before the asset enters Git history.
